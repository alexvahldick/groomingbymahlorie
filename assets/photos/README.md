# Photo Drop Zone

Put website-ready images here.

Recommended structure:

- `assets/photos/hero/` for replacement hero images.
- `assets/photos/grooms/` for completed groom gallery images.

Use optimized `.jpg` or `.webp` files around 1600-2200px wide for hero images and 900-1400px wide for gallery images. Keep original full-resolution phone photos outside the repo, then copy edited/exported versions here.
